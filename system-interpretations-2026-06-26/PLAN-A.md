- **Reductions** (CANON): sparse memory (M = N·h + K·b + S), Infinite-Three recursion (~1.5·B), never-explode resident bound (≤2000), tail-O(1) re-requests.
- **Bilateral vantage governance** (MEASURED on method): two seats build independently, read their own fabric, reconcile only via GitHub byte-hash; neither silently upgrades the other's claim. Everything live-touching is operator/cosign-gated.
