## Important formula boundaries

### π distortion

```text
π̂ = 4 · N(x²+y²≤1) / N
Δπ = |π̂−π|
```

`Δπ` is tested as a distribution/isotropy diagnostic. It is **not assumed** to be a universal
redundancy meter. Lens 3 measures its correlation with actual lossless bpc; a weak correlation is a
legitimate falsification of the stronger claim.

### Delay/decorrelation

```text
τ* = min{τ : I(X_t;X_(t+τ)) ≤ I_permutation + ε}
```

The plugin estimator has finite-sample bias, so a permutation baseline is retained.

### Blindness

```text
P(all k miss) = p^k
```

only under independent detector errors. Zero observed misses are reported as finite-trial upper
bounds, not exact zero probability.

### Zero container

Each of twenty lenses contributes three linear measurements over `F_65537`:

```text
dim Z_k = 60 − rank(A_k)
```

The rows form a 60×60 Vandermonde system by lens 20, so the final nullspace must be zero and the 60D
selector must be exactly solvable. This is a theorem instance, not a physical flashlight.

### Conditional coding

A receiver with correlated side information `Y` receives a compressed exact residual:

```text
D = X xor Y
X = Y xor D
```

Its wire rate may be below standalone `H(X)`, but `Y` remains paid in the civilization ledger.

### Secret sharing

```text
A=K
B=X xor K
X=A xor B
```

Software shares are copyable. This is a classical information-theoretic opacity test, not a physical
single-use quantum key.
